# AltOpsLab Landing Page

A clean, minimal, professional React landing page for AltOpsLab, a boutique service that consolidates and cleans alternative investment statements.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS

## Features

- Fully responsive design
- Clean, minimal, professional aesthetic
- Sticky header navigation
- Mobile-first approach
